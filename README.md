# Specular Differentiation

This repository contains the Python package `specular_diff` and codes for applications:

* **Initial value problems for ordinary differential equations**
  * Directory: `numerical-ODE/`
  * Related reference: [[2]](#references)

* **Nonsmooth convex optimization**
  * Directory: `nonsmooth-convex-opt/`
  * Related reference: [[3]](#references)


## References

[2] K. Jung and J. Oh. [The wave equation with specular derivatives](https://arxiv.org/abs/2210.06933). *arXiv preprint arXiv:2210.06933*, 2022.

[3] K. Jung and J. Oh. [Nonsmooth convex optimization using the specular gradient method with root-linear convergence](https://arxiv.org/abs/2412.20747). *arXiv preprint arXiv:2210.06933*, 2024.
